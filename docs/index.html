Secuencia

Proyecto: Comparador de secuencias.

Esta clase es la clase de la que heredarán todas los tipos de secuencia. Las secuencias han de tener como mínimo un método secuencia() que devuelva la secuencia como un string. Cada nueva secuencia se guardará en la variable global Secuencia.type (Secuencia.type.ejemplo) añadiendo el nombre siempre en minuscula. Cada operación nueva se guardará en la variable global Secuencia.operaciones siguiendo el mismo patron que para la secuencia.
